# Project PANOPTES Landing Site

This repository contains the source for the main [Project PANOPTES](https://panoptes.github.io/) landing site and documentation dashboard.

## Documentation Links

- [panoptes-utils](https://panoptes.github.io/panoptes-utils/)
- [POCS](https://panoptes.github.io/POCS/)

## Development

This site is built using [MkDocs](https://www.mkdocs.org/) and the [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) theme.

To run locally:

```bash
uv sync --group docs
uv run mkdocs serve
```
